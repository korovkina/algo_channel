## О проекте

Это рабочее пространство для планирования и создания контента для канала про алгоритмы. Проект использует данные из образовательного курса `/Users/tanya/Projects/Shortcut/algo`, но ведется отдельно как личная инициатива.

## Структура проекта

```
AlgoChannel/
├── content-plan/          # Планирование контента
│   ├── monthly-plans/     # Помесячные планы публикаций
│   ├── post-ideas.md      # Банк идей для постов
│   └── statistics.md      # Статистика и аналитика
├── posts/                 # Контент постов
│   ├── draft/            # Черновики
│   ├── published/        # Опубликованные посты
│   └── scheduled/        # Запланированные к публикации
├── analytics/            # Аналитика канала
│   ├── engagement.md     # Вовлеченность аудитории
│   └── growth.md         # Рост аудитории
└── assets/               # Ресурсы
    ├── images/          # Изображения для постов
    └── templates/       # Шаблоны постов
```

## Источники данных

### Курс алгоритмов (Школа)
**Путь:** `/Users/tanya/Projects/Shortcut/algo`

**Доступные ресурсы:**
- 15 лекций по алгоритмам и структурам данных
- Практические задания с LeetCode

**Использование:**
- Можно брать статистику по задачам
- Ссылаться на темы и паттерны из лекций
- Использовать задачи из практик для создания контента
- НЕ копировать целые лекции (это учебный материал школы)

## Принципы работы

1. **Контент от себя:** Канал ведется от личного имени, стиль и формат — мои
2. **Данные от школы:** Использую статистику, задачи, темы курса как источник
3. **Добавленная ценность:** Каждый пост должен давать что-то новое (мнение, инсайт, практику)
4. **Прозрачность:** Упоминаю школу там, где это уместно (амбассадор)

## Типы контента

- 📊 **Статистика и тренды** — разбор частых задач на собеседованиях
- 💡 **Разборы задач** — пошаговые объяснения решений
- 🎯 **Паттерны и техники** — обучающий контент по алгоритмам
- 🚀 **Советы по подготовке** — как готовиться к интервью
- 📈 **Личный опыт** — истории из практики, инсайты
