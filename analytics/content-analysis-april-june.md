## 📈 Что работает

### ✅ Успешные форматы

1. **Теория с глубоким объяснением**
   - Хеш-мапы (11 ❤️)
   - Префиксы (9 ❤️)
   - Сортировки (10 👍)
   - **Почему:** Объясняешь "почему", а не только "как"

2. **Свежие задачи с реальных собесов**
   - Указание компании и даты
   - Формат: условие → подсказка в спойлере → разбор через 3 часа
   - **Почему:** Актуальность + интерактивность

3. **Инсайты о процессе найма**
   - Пост про Яндекс (11 ❤️)
   - Статистика по темам (6 🔥)
   - **Почему:** Эксклюзивная информация

4. **Личный стиль**
   - Неформальный тон ("Че, пацаны и дамы")
   - Эмодзи для структуры
   - Обращения к читателю
   - **Почему:** Создает связь с аудиторией

### 📊 Паттерны вовлеченности

**Лучшее время публикации (по факту):**
- Утро (7:00-12:00) - большинство постов
- Задачи обычно утром, разборы днем

**Хештеги (наиболее используемые):**
- #задача, #разбор - для задач
- #теория - для объяснений
- #яндекс, #озон, #авито, #wildberries - компании

**Структура успешных постов:**
1. Хук (эмодзи + жирный заголовок)
2. Основной контент (с нумерацией через эмодзи)
3. Примеры/код
4. CTA (призыв к действию)

## 💡 Наблюдения

### Сильные стороны

1. **Уникальные данные**
   - 100+ реальных задач с собеседований 2025-2026
   - Упоминание: "У меня есть очень ценные закрома"

2. **Экспертность**
   - Работа в Яндексе
   - Автор курса в Шорткат
   - Личный опыт: "несколько лет работала в Яндексе"

3. **Регулярность**
   - Активный период: ежедневные посты 3+ недели
   - Стабильный формат

4. **Практичность**
   - Конкретные задачи, не абстрактная теория
   - Код с объяснениями
   - Подсказки в спойлерах

### Точки роста

1. **Визуальный контент**
   - Пока только один пост с картинкой (10 июня - мем)
   - Можно: инфографика, схемы, диаграммы

2. **Серийность**
   - Начали темы (строки, хеш-мапы), можно продолжить
   - Пример: "Хеш-мапы часть 2: коллизии" (было обещано)

3. **Кросс-ссылки**
   - Начали делать (ссылки на прошлые посты)
   - Можно: карта контента, навигация по темам

4. **Интерактивность**
   - Упражнения работают (но мало откликов)
   - Можно: челленджи, конкурсы

## 🎨 Стиль и подход

### Tone of Voice

**Характеристики:**
- Дружелюбный, но экспертный
- Неформальный ("пацаны и дамы", "м?")
- Поддерживающий ("Приходите ко мне, мне кайф с вами работать")
- Мотивирующий ("регулярная практика это самый эффективный способ")
- Местами строгий ("Слежу за вами", про практику)

**Структура постов:**
- Эмодзи для разделения (🥇, 1️⃣, 2️⃣)
- Жирный текст для акцентов
- Блоки кода в `<pre>`
- Спойлеры для подсказок
- Ссылки на LeetCode и внешние ресурсы

### Примеры удачных хуков

- "🔥 Врываюсь в эту неделю..."
- "Че, пацаны и дамы, хеш-мапы?"
- "❌ Частая ошибка:"
- "📊 Что РЕАЛЬНО спрашивают..."
- "❗️ Свежая задача из..."
