Часть 1: Классическая задача (LeetCode 20)

Паттерн: стек

Ключевая идея: используем стек для отслеживания открывающих скобок. Когда встречаем закрывающую скобку, проверяем, что она соответствует последней открывающей скобке (вершине стека).

Алгоритм:
1️⃣ Создаём пустой стек и словарь пар скобок
2️⃣ Проходим по строке слева направо:
   • Если открывающая скобка — кладём в стек
   • Если закрывающая скобка:
     - Проверяем, что стек не пуст
     - Проверяем, что вершина стека — соответствующая открывающая
     - Если да — удаляем из стека (pop)
     - Если нет — возвращаем false
3️⃣ В конце проверяем, что стек пуст (все скобки закрыты)

Код на Python:
def isValid(s: str) -> bool:
    # Словарь пар: закрывающая -> открывающая
    pairs = {')': '(', ']': '[', '}': '{'}

    # Стек для хранения открывающих скобок
    stack = []

    for char in s:
        if char in '([{':
            # Открывающая скобка — кладём в стек
            stack.append(char)
        else:
            # Закрывающая скобка
            # Проверяем, что стек не пуст
            if not stack:
                return False

            # Проверяем, что вершина стека соответствует
            if stack[-1] != pairs[char]:
                return False

            # Удаляем последнюю открывающую
            stack.pop()

    # Проверяем, что все скобки закрыты
    return len(stack) == 0

Пошаговый пример для "([)]":

Позиция 0: '(' → stack = ['(']
Позиция 1: '[' → stack = ['(', '[']
Позиция 2: ')' → вершина '[', ожидаем '(' → НЕ СОВПАДАЕТ → false

Сложность: O(n) времени, O(n) памяти
