⬇️ ТЕОРИЯ ⬇️

О стеке выше мы уже поговорили, теперь рассмотрим его вариацию для продвинутых задач, типа этой, — монотонный стек. 

❗️Монотонный стек — это стек, элементы которого расположены в отсортированном порядке.

Построим, например, монотонный стек с возрастающим порядком элементов. При добавлении нового элемента действуем так: 
1️⃣ Поочерёдно сравниваем его с уже имеющимися элементами, расположенными наверху стека.
2️⃣ Те из них, которые оказались больше его по значению, удаляются.
3️⃣ Элемент добавляется на вершину стека.

Пример: массив [2, 5, 8, 3].
1. Добавляем 2 -> [2]
2. Добавляем 5 -> [2, 5]
3. Добавляем 8 -> [2, 5, 8]
4. Добавляем 3 -> [2, 5, 8] (Pop 8, Pop 5) → [2, 3]

Стек с убывающим порядком работает аналогичным образом, только наоборот: удаляем меньшие элементы.

Со знанием этой концепции задача выше становится не Medium, а очень даже Easy. Так что дерзайте!
