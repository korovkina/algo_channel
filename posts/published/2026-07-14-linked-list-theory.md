Давайте разбираться со второй версией задачи на слияние — слияние двух связных (не двусвязных!) списков.

🐥 Материал подходит для новичков

Для начала: а что вообще такое связный список? По сути, это последовательность элементов, каждый из которых указывает на следующий (мем_со_спайдерменами.jpg). Эти элементы называются нодами. В каждой ноде есть:

1️⃣ Значение (любое, но обычно это инты)
2️⃣ Ссылка на следующую ноду

Выглядит это примерно так:

```text
head
 │
 ▼
[1|•]──▶[2|•]──▶[3|•]──▶[4|/]
```
Здесь `head` — ссылка на первую ноду, каждая `•` — ссылка на следующую, а `/` в последней ноде означает, что дальше ничего нет (`None` / `null`).

Ключевое отличие от массива: у нас есть непосредственный доступ только к голове связного списка, то есть к первому элементу. Чтобы взять любой другой элемент, нам нужно по этим самым ссылкам пройтись от начала списка до этого элемента.

Это как если бы вы играли в квест, где на каждой локации вас ждала бы записка, куда идти дальше. Вот тут то же самое.

Во многих языках готовая реализация есть в стандартной библиотеке (в Java это `LinkedList`, в C++ — `std::list`), но в Python встроенного связного списка нет. Да и на собеседованиях по алгоритмам библиотечный список обычно не используют: список либо уже дан в условии, либо его просят собрать руками из нод — так что полезно уметь делать это самому. Чтобы сделать связный список, достаточно создать объект ноды, а затем наплодить их столько, сколько вам угодно, с какими-то значениями и ссылками друг на друга.

Объект ноды может выглядеть, например, так:
```python
class Node:
    def __init__(self, value, next=None):
        self.value = value
        self.next = next
```
```java
class Node {
    int value;
    Node next;

    Node(int value) {
        this.value = value;
        this.next = null;
    }
}
```

Двусвязный список, упомянутый в заголовке — это все то же самое, но указатель есть не только на следующую ноду (next), но и на предыдущую (prev), и ходить по нему можно в две стороны. Такие списки на собесах попадаются крайне редко, и вообще это зверь особый, заслуживающий отдельного поста.
