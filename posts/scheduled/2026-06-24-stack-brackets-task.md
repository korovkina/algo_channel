🔥 Задача из rtlab (Госуслуги)

**Сложность:** Easy → Medium (с бонусом)

**Валидные скобки (Valid Parentheses)**

Дана строка, состоящая только из символов `'('`, `')'`, `'['`, `']'`, `'{'`, `'}'`.

Определите, является ли эта строка валидной скобочной последовательностью.

**Правила:**
• Каждая открывающая скобка должна быть закрыта скобкой того же типа
• Открывающие скобки должны закрываться в правильном порядке
• Каждая закрывающая скобка должна иметь соответствующую открывающую скобку того же типа

**Требования:**
• Время: O(n)
• Память: O(n)

**Примеры:**

Вход: `"()"`
Выход: `true`

Вход: `"()[]{}"`
Выход: `true`

Вход: `"(]"`
Выход: `false`
_Объяснение: Открывающая `(` закрывается неправильной скобкой `]`._

Вход: `"([)]"`
Выход: `false`
_Объяснение: `[` открыта последней, но закрывается не первой._

Вход: `"{[]}"`
Выход: `true`
_Объяснение: Все скобки правильно вложены._

## Часть 1: Классическое решение (LeetCode 20)

**Паттерн:** Стек

**Ключевая идея:** Используем стек для отслеживания открывающих скобок. Когда встречаем закрывающую скобку, проверяем, что она соответствует последней открывающей скобке (вершине стека).

**Алгоритм:**

1️⃣ Создаём пустой стек и словарь пар скобок

2️⃣ Проходим по строке слева направо:
   • Если открывающая скобка — кладём в стек
   • Если закрывающая скобка:
     - Проверяем, что стек не пуст
     - Проверяем, что вершина стека — соответствующая открывающая
     - Если да — удаляем из стека (pop)
     - Если нет — возвращаем `false`

3️⃣ В конце проверяем, что стек пуст (все скобки закрыты)

**Код на Python:**

```python
def isValid(s: str) -> bool:
    # Словарь пар: закрывающая -> открывающая
    pairs = {')': '(', ']': '[', '}': '{'}

    # Стек для хранения открывающих скобок
    stack = []

    for char in s:
        if char in '([{':
            # Открывающая скобка — кладём в стек
            stack.append(char)
        else:
            # Закрывающая скобка
            # Проверяем, что стек не пуст
            if not stack:
                return False

            # Проверяем, что вершина стека соответствует
            if stack[-1] != pairs[char]:
                return False

            # Удаляем последнюю открывающую
            stack.pop()

    # Проверяем, что все скобки закрыты
    return len(stack) == 0
```

**Пошаговый пример для `"([)]"`:**

Позиция 0: `'('` → stack = `['(']`
Позиция 1: `'['` → stack = `['(', '[']`
Позиция 2: `')'` → вершина `'['`, ожидаем `'('` → **НЕ СОВПАДАЕТ** → `false`

**Сложность:** O(n) времени, O(n) памяти

## Часть 2: Бонусная модификация — порядок не важен

**Как на собеседовании:** Интервьюер говорит: "Отлично! А теперь измените решение так, чтобы порядок скобок был не важен."

**Ключевое изменение:** Вместо одного стека используем **три отдельных стека** — для каждого типа скобок.

**Почему это работает:**
- Теперь каждая закрывающая скобка ищет соответствующую открывающую только среди своего типа
- Порядок между разными типами скобок не важен
- Но внутри одного типа порядок всё равно соблюдается (LIFO)

**Минимальное изменение кода:**

```python
def isValidUnordered(s: str) -> bool:
    # ТРИ СТЕКА вместо одного!
    stacks = {
        '(': [],  # стек для круглых скобок
        '[': [],  # стек для квадратных скобок
        '{': []   # стек для фигурных скобок
    }

    pairs = {')': '(', ']': '[', '}': '{'}

    for char in s:
        if char in '([{':
            # Открывающая — кладём в соответствующий стек
            stacks[char].append(char)
        else:
            # Закрывающая — проверяем соответствующий стек
            matching_open = pairs[char]

            # Проверяем, что в нужном стеке есть элементы
            if not stacks[matching_open]:
                return False

            # Удаляем из соответствующего стека
            stacks[matching_open].pop()

    # Проверяем, что все три стека пусты
    return (len(stacks['(']) == 0 and
            len(stacks['[']) == 0 and
            len(stacks['{']) == 0)
```

**Пошаговый пример для `"([)]"`:**

Позиция 0: `'('` → stacks = `{'(': ['('], '[': [], '{': []}`
Позиция 1: `'['` → stacks = `{'(': ['('], '[': ['['], '{': []}`
Позиция 2: `')'` → ищем в стеке `'('`, он не пуст → pop → stacks = `{'(': [], '[': ['['], '{': []}`
Позиция 3: `']'` → ищем в стеке `'['`, он не пуст → pop → stacks = `{'(': [], '[': [], '{': []}`

Все стеки пусты → `true` ✅

**Сложность:** O(n) времени, O(n) памяти (так же, как классическое решение)

——

**Что изменилось:**
- Было: `stack = []` (один стек для всех)
- Стало: `stacks = {'(': [], '[': [], '{': []}` (три стека)

- Было: `stack.append(char)` для всех открывающих
- Стало: `stacks[char].append(char)` для каждого типа отдельно

- Было: `if stack[-1] != pairs[char]` (проверяем вершину общего стека)
- Стало: `if not stacks[matching_open]` (проверяем соответствующий стек)
